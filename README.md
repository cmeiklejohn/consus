# Consus

Lighweight prototype Dropbox clone written for Riak CS.

## Configuration.

* Configure ```config/riak_cs.yml``` with your Riak CS configuration.
* Configure ```config/database.yml``` with your database configuration.
* ```bundle```
* ```foreman start```
